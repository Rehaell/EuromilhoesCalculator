EuromilhoesCalculator
=====================

- BETA -

Console version

Given two array of numbers with the lengths 5 and 2, calculate how much time in years, drafts and money spent will a average person
need to win a game of Euromillions.

The program tests two combinations : 
 	- 5 number main pick
 	- 2 number secondary pick
 
 
 The first prize is achieved when the user matches de 5 + 2 numbers choosen with the ones draft. 
 When this is achieved, the program will end returning the amount of time and money (€2 per draft) spent to achieve this result.
 
 In this first test, other prize combinations aren't tested.
